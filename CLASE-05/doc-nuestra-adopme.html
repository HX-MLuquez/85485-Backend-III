🐾 AdoptMe API

Documentación completa de la API REST para el sistema de adopción de mascotas

Información General

Esta API proporciona endpoints para gestionar usuarios, mascotas, adopciones y sesiones en la plataforma AdoptMe.

Base URL: http://localhost:8080/api
GET /users

Obtiene la lista de todos los usuarios registrados.

Respuesta exitosa
200Array de usuarios
{"status":"success","payload":[{"_id":"507f1f77bcf86cd799439011","first_name":"Juan","last_name":"Pérez","email":"juan@example.com","role":"user","pets":[]}]}
GET /users/:uid

Obtiene información de un usuario específico.

Parámetros de ruta
uid string REQUERIDO
ID único del usuario
PUT /users/:uid

Actualiza información de un usuario.

Body (JSON)
first_name string OPCIONAL
last_name string OPCIONAL
email string OPCIONAL
DELETE /users/:uid

Elimina un usuario del sistema.

GET /pets

Obtiene todas las mascotas disponibles.

POST /pets

Crea una nueva mascota sin imagen.

Body (JSON)
name string REQUERIDO
specie string REQUERIDO
POST /pets/withimage

Crea una mascota con imagen.

Body (multipart/form-data)
image file REQUERIDO
PUT /pets/:pid

Actualiza una mascota.

DELETE /pets/:pid

Elimina una mascota.

GET /adoptions

Obtiene todas las adopciones.

GET /adoptions/:aid

Obtiene una adopción específica.

POST /adoptions/:uid/:pid

Crea una nueva adopción.

Parámetros de ruta
uid string REQUERIDO
ID del usuario
pid string REQUERIDO
ID de la mascota
POST /sessions/register

Registra un nuevo usuario.

Body (JSON)
first_name string REQUERIDO
last_name string REQUERIDO
email string REQUERIDO
password string REQUERIDO
POST /sessions/login

Inicia sesión.

Body (JSON)
email string REQUERIDO
password string REQUERIDO
GET /sessions/current

Obtiene el usuario actual.
